O que é um algoritmo

Um algoritmo é um processo sistemático para a resolução de um problema. Ele descreve, de forma lógica, os passos a serem executados no cumprimento de uma determinada tarefa. Um algoritmo pode ser uma descrição narrativa, um fluxograma ou um pseudocódigo.

Independência de máquina e linguagem

Os algoritmos são independentes da máquina e da linguagem de programação utilizada. Eles podem ser realizados tanto por uma máquina quanto por um humano. Um programa é o resultado da codificação de um algoritmo em uma linguagem de programação específica.

Qualidades de um algoritmo correto

Para um algoritmo ser considerado correto, ele deve possuir três qualidades principais: cada passo deve ser uma instrução realizável, a ordem dos passos deve ser precisamente determinada e o algoritmo deve ter um fim.

Tipos de algoritmos

Existem vários tipos de algoritmos, sendo os mais comuns a descrição narrativa, o fluxograma e o pseudocódigo.
