Inteiros

São números inteiros, assim como na matemática, não podendo assumir valores fracionários

Como declarar em c

int nome_da_variável;

Sendo possível inicializá-la ainda na declaração:

int nome_da_variável = valor;

Exemplo de declaração de um número inteiro

int idade;

Declaração com Inicialização

int idade = 18;

Como operar com números inteiros

Toda operação tem que estar relacionada à alguma variável, atribuindo o resultado da operação a ela

Exemplo:

int idade = 18;

idade = idade + 2;

Neste exemplo a variável que antes possuia valor 18, foi atualizada recebendo o valor de quanto ela valia(18) somado com 2, logo o valor armazenado na variável idade agora é 20
